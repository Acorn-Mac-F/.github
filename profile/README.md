## Acorn Mac: Layers and Non-Destructive Editing

Layered, non-destructive editing is the backbone of the <a href="#">Acorn app Mac</a>, and it is
what separates a real image editor from a simple photo viewer:

### Layer-Based Composition

Every element in an Acorn document can live on its own layer — photos, shapes, text, and
adjustments stacked independently. Layers can be reordered, hidden, grouped, and blended, letting
you build complex composites where each piece stays editable and separate from the rest. This is the
same layer model professionals expect, implemented cleanly in <a href="#">Acorn Mac</a>.

### Non-Destructive Filters

Filters and adjustments in Acorn are applied non-destructively, meaning the original pixels are
never permanently altered. You can stack, reorder, tweak, and remove effects at any time without
degrading the source image. This freedom to experiment and revise is central to a modern editing
workflow and a highlight of the Acorn app.

### Blend Modes and Masks

Blend modes control how layers combine, enabling lighting effects, color grading, and creative
compositing. Layer masks let you reveal and hide parts of a layer with brushed precision, so you can
blend images seamlessly — a core technique for photo manipulation in <a href="#">Acorn Mac</a>.

## Key Terms

**Layer** — an independent level in an image document holding one element (photo, shape, text, or
adjustment) that can be edited separately from the rest.

**Non-destructive editing** — a workflow where filters and adjustments never permanently alter the
original pixels, so every change stays editable and reversible.

**Raster** — pixel-based imagery (photos), as opposed to vector; the primary editing domain of the
Acorn app.

**Vector** — resolution-independent shapes defined by paths that scale without quality loss, used for
logos and icons.

**Blend mode** — a setting that determines how a layer's colors combine with the layers beneath it.

**Layer mask** — a grayscale overlay that hides or reveals parts of a layer for seamless compositing.
